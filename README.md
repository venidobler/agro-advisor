# 🌾 Agro Advisor 2.0

> Plataforma inteligente para monitoramento de commodities e análise preditiva agrícola.

## 🚀 Sobre o Projeto
O **Agro Advisor** é uma solução Full-Stack desenvolvida para centralizar dados do agronegócio, consumindo APIs públicas de cotações e clima para gerar insights através de modelos de Machine Learning.

Este projeto é uma evolução arquitetural completa de uma versão anterior, focando agora em escalabilidade, resiliência de dados (anti-scraping) e padrões de design modernos.

## 🛠️ Tecnologias
- **Backend:** Python, Django REST Framework, Scikit-learn.
- **Frontend:** React (Vite), Tailwind CSS, Shadcn/ui, Highcharts.
- **Database:** PostgreSQL.
- **Infra:** Docker & Docker Compose.
