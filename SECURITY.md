# Política de Segurança

## Versões Suportadas
Atualmente, apenas a versão principal (`main`) recebe atualizações de segurança.

| Versão | Suportada          |
| -------| ------------------ |
| 2.0.x  | :white_check_mark: |
| < 2.0  | :x:                |

## Como reportar uma vulnerabilidade
Por favor, **não abra uma issue pública** para reportar vulnerabilidades de segurança críticas. 

Se você descobrir um problema de segurança no Agro Advisor (ex: vazamento de chaves de API de cotação ou exposição do banco de dados), por favor, envie um e-mail diretamente para o mantenedor do projeto com os detalhes da falha e os passos para reproduzi-la. 

Nós nos esforçamos para resolver todos os problemas de segurança validados o mais rápido possível.
